Ejercicio 1 · POO básica y estructuras de control

Clase, atributos privados, constructor, getters/setters, if-else y bucle for
2,5 puntos

Crea la clase Libro con los atributos privados titulo (String), autor (String), anyo (int) y precio (double).

  1. a) Constructor con los cuatro parámetros. Getters y setters para cada atributo. [1,0 pt]
  2. b) Método mostrarFicha() que imprima: [año] Título — Autor (X,XX €). [0,5 pts]
  3. c) En el main: crea dos objetos Libro y usa if-else para mostrar cuál es más reciente. Si son iguales, imprime "Misma edición". [0,5 pts]
  4. d) Con un bucle for, muestra el precio de un libro aplicando descuentos del 5 %, 10 %, 15 % y 20 %, indicando el porcentaje en cada línea. [0,5 pts]

Ejercicio 2 · Interfaz Catalogable y herencia

Interfaz, implementación, herencia, sobrescritura y polimorfismo
2,5 puntos

El sistema de la biblioteca necesita estandarizar cómo se muestran los objetos en el catálogo.

  1. a) Define la interfaz Catalogable con dos métodos:
    • String getCodigo()
    • void mostrarEnCatalogo()
    (Nota: mostrarEnCatalogo() imprime por consola, no devuelve String) [0,5 pts]
  2. b) Haz que Libro implemente Catalogable.
    • El código se forma concatenando "LIB-", el año y las tres primeras letras del título en mayúsculas.
    • El método mostrarEnCatalogo() llamará a mostrarFicha() y añadirá el código entre corchetes al final.
    [2020] ... (9,99 €) [LIB-2020-SOL]
  3. c) Crea LibroDigital que herede de Libro y añada el atributo privado formato (String, p. ej. "EPUB"). Añade constructor con super(), getter/setter y sobreescribe mostrarEnCatalogo() para mostrar también el formato. [0,75 pts]
  4. d) En el main: crea un Libro y un LibroDigital, guárdalos en un array Catalogable[] y recórrelo llamando a mostrarEnCatalogo(). Añade un comentario explicando qué concepto OOP demuestra. [0,5 pts]

Ejercicio 3 · Interfaz OperableMatriz y herencia

Matrices, interfaz, clase base, herencia y polimorfismo
2,5 puntos

El sistema necesita estandarizar cómo se trabajan las matrices de datos.

  1. a) Define la interfaz OperableMatriz con dos métodos:
    • int[][] sumar(int[][] otra)
    • void mostrarResultado()
    (Nota: mostrarResultado() imprime por consola, no devuelve String) [0,5 pts]
  2. b) Crea la clase MatrizDatos con el atributo privado datos (int[][]). Añade constructor, getter y setter. Haz que implemente OperableMatriz.
    • El método sumar(int[][] otra) devolverá una nueva matriz resultado de sumar ambas matrices.
    • Si otra es null o no tiene las mismas dimensiones, lanzará IllegalArgumentException.
    • El método mostrarResultado() imprimirá la matriz por consola.
    [1,0 pt]
  3. c) Crea MatrizCuadrada que herede de MatrizDatos. Sobrescribe mostrarResultado() para añadir al final el tamaño de la matriz, por ejemplo:
    Matriz cuadrada: 3 x 3
    [0,5 pts]
  4. d) En el main: crea un objeto MatrizDatos y un objeto MatrizCuadrada, guárdalos en un array OperableMatriz[] y recórrelo llamando a mostrarResultado(). Añade un comentario explicando qué concepto OOP demuestra. [0,5 pts]

Ejercicio 4 · Comparable y Comparator

Orden natural, comparadores externos y ordenación de arrays de objetos
2,5 puntos

El sistema necesita ordenar los libros de diferentes maneras.

  1. a) Haz que la clase Libro implemente la interfaz Comparable<Libro> para definir el orden natural por título en orden alfabético. Implementa el método compareTo(Libro otro). [0,75 pts]
  2. b) Crea la clase ComparadorPorAnyo que implemente Comparator<Libro>. Debe permitir ordenar los libros por año de publicación, del más antiguo al más reciente. [0,5 pts]
  3. c) Crea la clase ComparadorPorPrecio que implemente Comparator<Libro>. Debe permitir ordenar los libros por precio, de menor a mayor. [0,5 pts]
  4. d) En el main:
    • Crea un array de al menos 4 objetos Libro.
    • Ordénalo primero usando el orden natural (Comparable).
    • Después ordénalo por año usando ComparadorPorAnyo.
    • Finalmente ordénalo por precio usando ComparadorPorPrecio.
    • Muestra el contenido del array tras cada ordenación usando mostrarFicha().
    [0,5 pts]
  5. e) Añade un comentario explicando qué diferencia hay entre Comparable y Comparator. [0,25 pts]
