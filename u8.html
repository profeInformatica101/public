UT8: Generación Dinámica de Páginas Web con Tecnologías y Frameworks - OpenCMS y Spring Boot

FP DUAL Del 10 de febrero al 14 de marzo

Descripción

En esta unidad, los estudiantes, organizados en equipos de (3-5), colaborarán con una empresa real o ficticia para desarrollar una solución web utilizando tecnologías como Spring Boot, Thymeleaf, y/o OpenCMS. El proyecto implicará la creación de un sistema de gestión de contenidos dinámicos, con enfoque en la verificación de formularios, interacción con el usuario y la modificación dinámica de contenido.

Los estudiantes trabajarán en un entorno simulado o real, desarrollando soluciones técnicas para la empresa, además de aplicar soft skills como la colaboración, la gestión del tiempo, y la comunicación efectiva.

Resultado de Aprendizaje

8. Genera páginas web dinámicas analizando y utilizando tecnologías y frameworks del servidor web que añaden código al lenguaje de marcas.

Criterios de Evaluación

  • a) Se han identificado las diferencias entre la ejecución de código en el servidor y en el cliente web.
  • b) Se han reconocido las ventajas de unir ambas tecnologías en el proceso de desarrollo de programas.
  • c) Se han identificado las tecnologías y frameworks relacionadas con la generación por parte del servidor de páginas web con guiones embebidos.
  • d) Se han utilizado estas tecnologías y frameworks para generar páginas web que incluyan interacción con el usuario.
  • e) Se han utilizado estas tecnologías y frameworks, para generar páginas web que incluyan verificación de formularios.
  • f) Se han utilizado estas tecnologías y frameworks para generar páginas web que incluyan modificación dinámica de su contenido y su estructura.
  • g) Se han aplicado estas tecnologías y frameworks en la programación de aplicaciones web.

Actividades

Semana 1-2: Definición del Proyecto y Configuración

  • Reunión con la empresa para definir los requisitos del proyecto.
  • Configuración de OpenCMS (y/o) Spring Boot.
  • Planificación y reparto de tareas dentro del equipo utilizando GitHub.
  • Soft Skills: Comunicación y planificación en equipo.

Semana 3-4: Desarrollo del Proyecto

  • Desarrollo de las funcionalidades clave del proyecto (formularios, vistas dinámicas, interacción con el usuario).
  • Integración de Spring Boot con OpenCMS para gestionar contenidos.
  • Revisión periódica con la empresa para obtener retroalimentación.
  • Soft Skills: Resolución de problemas en equipo y manejo de conflictos.

Semana 5: Finalización y Presentación

  • Finalización del desarrollo y pruebas de la aplicación.
  • Presentación del proyecto a la empresa y retroalimentación final.
  • Soft Skills: Presentación clara y estructurada del proyecto.

Prueba de Evaluación

Los estudiantes serán evaluados en dos aspectos:

  • Parte técnica: Desarrollo del sistema web completo con Spring Boot, Thymeleaf, y OpenCMS.
  • Soft Skills: Evaluación de la colaboración en equipo, el uso de GitHub, y la comunicación con la empresa.

Requisitos para la Entrega

  • Repositorio en GitHub con todo el código fuente y documentación del proyecto.
  • Presentación del proyecto a la empresa con demostración de las funcionalidades implementadas.
